# 李涛简历网站（RAG / Agent 方向）

## 运行方式

这是纯前端静态站点，无需安装依赖。

1. 直接双击打开 `index.html`
2. 或在该目录执行任意静态服务器（推荐）

示例（Python）：

```bash
python -m http.server 8080
```

然后访问：`http://localhost:8080`

## 文件说明

- `index.html`：页面结构与内容
- `styles.css`：视觉风格、响应式、动画样式
- `script.js`：打字机、滚动显隐、导航高亮、粒子背景、3D 技能球
- `projects.html`：项目详情页面
- `curriculum.html`：课程学习页面
- `knowledge-map.html`：知识图谱页面

## 功能特性

### 已实现功能

- **明暗主题切换**：右上角按钮切换，支持 localStorage 持久化
- **3D 粒子背景**：Three.js 实现的交互式粒子网络
- **3D 技能球**：可拖拽旋转的 3D 技能展示组件
- **终端打字效果**：英雄区域命令行风格的技能展示
- **技能时间线**：展示 3 年 AI 工程经验成长路径
- **滚动进度条**：顶部彩色进度条显示页面浏览进度
- **PDF 导出**：点击页脚”下载 PDF”一键导出简历
- **响应式设计**：支持桌面端和移动端自适应

### 交互 Demo

- 内嵌 RAG 系统演示（需启动后端 API）
- 支持简单问答和 RAG 检索两种模式

## 快速修改内容

在 `index.html` 中搜索以下关键词并替换为你的真实信息：

- `电话、邮箱、城市可按你的真实简历补充`
- `3+ RAG 系统迭代`
- `10+ 关键模块封装`
- 各项目中的结果指标（如召回率、延迟、准确率）

## 技术栈

- **前端框架**：原生 HTML/CSS/JavaScript
- **3D 渲染**：Three.js
- **图表**：D3.js (知识图谱)
- **PDF 导出**：html2pdf.js
