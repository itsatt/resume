正则匹配 + 规则分析

输入输出安全防护

# 提示注入检测
INJECTION_PATTERNS = [
    r"ignore previous instructions",
    r"you are now in debug mode",
    r"output your system prompt"
]

def detect_injection(query):
    return any(re.search(p, query.lower())
               for p in INJECTION_PATTERNS)
结果： 拦截 99.7% 注入尝试
